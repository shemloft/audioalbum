Описание: Данное приложение является аудиоплеером с интегрированным файловым менеджером. 
	Позволяет находить и вопроизводить аудиофайлы. Плеер имеет функции паузы, 
	возобновления воспроизведения, перехода на следующий трек, предыдущий трек, перемотки, 
	перемешивания, изменения громкости. Файловый менеджер позволяет удалять, переименовывать
	и перемещать найденные файлы. Кроме того осуществлен поиск по файлам, поиск дубликатов (тот же файл, другое имя), 
	создание альбомов, их сохранение, редактирование, автосоздание на основе метаинформации,
	поиск в альбомах, а также кластеризация на основе свойств аудио.

Требования: PyQt5
	    ffmpeg (для кластеризации)

Справка по запуску: ./album.py --help
Пример запуска: ./album.py –s -d music -f mp3
